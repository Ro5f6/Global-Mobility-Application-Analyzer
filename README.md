# Global-Mobility-Application-Analyzer

## How to run?

1. Clone the repository:
   ```bash
   git clone https://github.com/Ro5f6/Global-Mobility-Application-Analyzer.git
   ```

2. Navigate to the project directory:
   ```bash
   https://github.com/Ro5f6/Global-Mobility-Application-Analyzer.git
   ```
   ```bash
   cd Global-Mobility-Application-Analyzer
   ```

3. Create and activate a virtual environment:
   ```bash
   conda create -n application python=3.8 -y
   ```
   ```bash
   conda activate application
   ```

4. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
